### アイテムマイセット自動適用Lua

- Luaの練習用(自分用)に作成しました。  
- マイセット装備に紐づいたアイテムマイセットを自動で適用します。  
- クエスト帰還後に適用するアイテムマイセットを選択できます。  
  => マイセット装備を選択した場合は以降紐づいたアイテムマイセットを適用します。  
- 設定を変更するとdataフォルダに設定ファイルを出力します。
- 特にちゃんとした動作確認はしていません。  
  あくまでも自分用なので日本語以外に対応するつもりはありません。
